ESLint

ESLint的定义

ESLint是一个代码风格约束工具，用于约束代码风格，能够帮助团队编写出代码风格一致的代码。
Find and fix problems in your JavaScript code

官网：https://eslint.org/

ESLint的安装

安装成功后会自动创建.eslintrc.js，用于配置ESLint。
